# 안티 페르소나 소환 기능 설계

- 목적: 어떤 인물(persona)에 대한 반대/대비되는 성격을 가진 가상의 새 인물(안티 페르소나)을 소환하여, 에이전트가 해당 인물로서 행동하도록 한다.
- 용도: 악역·데빌스 어드보킷·스트레스 테스트·롤플레이 등.

---

## 1. 정의

- **안티 페르소나**: 원본 인물(`name`)과 동일한 나이·사회적 위치·환경을 가지지만, 성격과 가치관이 정반대인 가상의 새 인물. 원본 인물이 역할을 맡는 것이 아니라 독립된 별개의 존재.
- **소환**: 원본 프로필과 소환 지침을 반환하면, LLM이 그를 바탕으로 반대 성격의 가상 인물을 즉석 구성하여 그 인물로서 응답하는 것.

---

## 2. 저장 방식

- **별도 저장 데이터 없음**: 원본 `profiles/{name}.md`를 읽어 LLM이 즉석으로 안티 인물을 도출.
- **DB**: 사용하지 않음.

---

## 3. 도구

- **이름**: `summon_anti_persona(name)`
- **동작**:
    1. `name`을 해석 (`@me` → `me`).
    2. `profiles/{name}.md`(원본 프로필) 전체를 로드.
    3. 소환 지침 + 원본 프로필을 반환. LLM이 반대 성격의 독립 인물을 구성.
    4. 프로필이 없으면 "프로필을 먼저 생성하세요" 안내 반환.
- **호출 시점**: 맥락상 안티 페르소나 소환이 적절하다고 LLM이 판단할 때.
